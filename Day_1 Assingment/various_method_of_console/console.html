Console Methods

we can use the console to debugging code.console shows the output in the browser's console

console.log();

  • console.log() is method used to execute the output in the browser console
  • message may include string or any object of javascript
    1. string
    2. object
    3. array
    4. number
    5. etc..

console.dir

  • The Console method dir() displays an interactive list of the properties of the specified JavaScript object.
    The output is presented as a hierarchical listing with disclosure triangles that let you see the contents of
    child objects

console.error();

  • used to Show up the error message in the browser console

console.warn();

  • Same as console.error() it will also shows the warn message in the browser Console

console.table();

  • Create a table with mutiple element in a user order

console.time();

  • time function is using to see a time,and we got a exact time in the console

console.info();

  • console.info is a method used to write a message to the Console
  • message may include a string,number,big int,array,object or etc

console.trace

  • The console interface's trace() method outputs a stack trace to the Web Console.

console.count

  • The console.count() method logs the number of times that
    this particular call to count() has been called.
